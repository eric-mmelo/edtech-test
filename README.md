🚀 Teste Técnico – Desenvolvedor Frontend (EdTech)
📖 Sobre o Projeto

Este projeto consiste na implementação completa de uma página baseada em um layout fornecido no Figma, com foco em fidelidade visual, responsividade e interatividade.

O desenvolvimento foi realizado utilizando apenas HTML5, CSS e JavaScript Vanilla, conforme os requisitos do teste técnico.

🛠 Tecnologias Utilizadas
-HTML5
-CSS3
-JavaScript (Vanilla)

🎯 Funcionalidades Implementadas

📱 Responsividade
-Layout adaptado para desktop e dispositivos móveis
-Componentes ajustáveis conforme diferentes tamanhos de tela

🎬 Player de Vídeo
-Embed de vídeo do YouTube
-Totalmente responsivo

🖼 Imagens
-Imagens adaptadas ao layout
-Responsivas

🎞 Slider de Imagens
-Slider funcional com no mínimo 3 imagens
-Estados implementados:
-Normal
-Hover
-Disabled

🔊 Player de Áudio
-Player funcional com controles básicos:
-Play / Pause
-Barra de progresso

🃏 Cards Interativos
-Expansão ao clicar em “Abrir”
-Recolhimento ao clicar novamente
-Controle de estado via JavaScript

📝 Atividades
Atividade Discursiva
-Campo de texto para resposta
Botão Responder:
-Exibe feedback
-Desabilita edição
Botão Alterar:
-Permite edição novamente

✅ Atividade Objetiva
-Seleção via checkbox
-Destaque visual ao selecionar
-Botões com controle de estado:
-Responder
-Alterar
-Feedback exibido conforme interação

💾 Persistência de Dados
Utilização de sessionStorage
Armazena:
-Respostas do usuário
-Estado dos botões
-Feedback exibido
-Opções selecionadas
-Dados restaurados automaticamente ao recarregar a página

📚 FAQ (Accordion)
-Implementação com JavaScript puro
-Comportamento:
-Expandir ao clicar
-Recolher ao clicar novamente
-Destaque visual no item ativo

✨ Diferenciais
-Transições suaves entre estados
-Microinterações
-Código organizado e semântico
-Boas práticas de responsividade

♿ Acessibilidade
-Uso de HTML semântico
-Navegação por teclado
-Estados de foco visíveis
-Estrutura preparada para leitores de tela
